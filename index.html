QWeather帮助页面

获取key&注册账号

1. 前往 https://id.qweather.com/#/register 注册qweather开发者账号
1.1 可以完成实名认证注册为个人开发者, 完成后可以获取7天的天气
2. 前往 https://dev.qweather.com/docs/resource/get-key/ 查看如何获取key
注: API key 类型为Web API
至此api key获取已完成

-----------------------------------------------------------------------------------

获取邮箱登录秘钥(以yeah.net为例)

你需要拥有一个yeah.net的邮箱账号
1. 前往 https://mail.yeah.net/ 并登陆
2. 依次点击: 设置-> POP3/SMTP/IMAP-> IMAP/SMTP 服务 已关闭|开启-> 继续开启-> (扫描二维码发送短信)-> 我已发送-> 复制显示的秘钥并粘贴到一个txt文件内
至此获取邮箱登录秘钥已完成

运行程序前准备(windows 为例)

下载项目本体
1.0 前往 https://github.com/MarkusJoe/QWeather/releases 下载最新的release(下载Source Code.zip)
1.1 解压到任何你想要的地方, 并打开它, 你可以看到 QWeather.py
安装python3.10.x+
2.0 https://www.python.org/downloads/release/python-3101/ 下载Windows installer(64-bit) 并安装
----------------------------------------------------------------------------------- 填写配置文件
3.0 你可以看到mail-settings, request-settings, client-settings, only-view-settings
注: 如果在填写配置文件时选项冒号后没有空格必须要手动添加一个空格才能输入数据
3.1 在mail-settings下找到sender 填写获取秘钥的邮箱账号(例如用xxx@yeah.net获取的秘钥, 就填写xxx@yeah.net)
3.2 在mail-settings下找到password 粘贴刚才在txt文件内粘贴的秘钥
3.3 在mail-settings下找到server 填写smtp.yeah.net
3.4 在request-settings下找到key 填写第一步申请的key秘钥
3.4.1 在request-settings下找到mode 如果你是免费版用户直接跳过此步， 如果是个人开发者请将free 改成 dev
3.5 在request-settings下找到receiver 填写接受邮件的邮箱可以是一个也可以是多个(不要填错)
3.6 在client-settings下找到send-times 并修改发送的时间, 可以自行添加或修改
其他选项不要随意更改
-----------------------------------------------------------------------------------
安装所需的库
4.0 (windows)打开QWeather文件夹 在空白处shift + 右键 选择 在此处打开powershell 输入 pip install -r requirements.txt 等待安装完成
4.1 (linux/ubuntu) 打开QWeather文件夹, 右键空白处选择在终端中打开, 输入pip3 install -r requirements.txt 等待安装完成
至此运行前准备已完成

-----------------------------------------------------------------------------------

运行项目

1.0 (windows)打开QWeather文件夹, 在空白处shift + 右键 选择 在此处打开powershell 输入 python QWeather.py
2.0 (linux/ubuntu)打开QWeather文件夹, 右键空白处选择在终端中打开, 输入python3 QWeather.py
3.0 首次运行QWeather.py 会有进行引导填写城市, 请根据提示完成填写
至此部署已完成
